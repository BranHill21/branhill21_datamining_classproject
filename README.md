# Graph Mining & Analysis: Citation Network Exploration

This repository contains a comprehensive Jupyter Notebook (`Project1_datamining.ipynb`) designed for large-scale graph analysis of academic citation networks, specifically utilizing the **ACM-Citation-network-V12** dataset. The analysis explores structural properties, identifies influential papers through centrality measures, and decomposes the network to uncover hidden community structures.

## Overview

The notebook implements a high-performance graph mining pipeline using the `igraph` library to handle over 6.6 million nodes and 13 million edges efficiently. It is structured to perform exploratory data analysis (EDA) and execute advanced graph algorithms within standard memory constraints.

### Key Features
* **Memory-Safe Parsing:** Custom streaming logic to load multi-gigabyte JSON citation data line-by-line while handling inconsistent schema types (e.g., `venue` as string or dictionary).
* **Centrality Comparisons:** Implementation of PageRank, HITS (Hubs and Authorities), and In-degree metrics to identify seminal research versus globally popular nodes.
* **Structural Connectivity:** Analysis of Weakly Connected Components (WCC) and Strongly Connected Components (SCC), including internal statistics (density, avg path length) for representative clusters.
* **Local Community Detection:** Use of Personalized PageRank (PPR) to extract and visualize thematic clusters around specific seed papers, compared against baseline Ego-networks.
* **Bow-tie Decomposition:** Partitioning of the graph into IN, OUT, and GSCC components to visualize the macro-level flow of academic influence and identify "citation sinks."
